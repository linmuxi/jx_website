薪金e代 产品说明书
基本特征
客户群体 针对受薪类客户群体
客户分类 A类：政府公务员、事业单位、国有企业 贷款额度 【3，15】万
B类：上市公司、外资500强企业、商业银行客户 贷款额度 【3，15】万
C类：一般单位受薪客户 贷款额度 【2，10】万
放款手续费 贷款金额*3% 贷款期限 12、15、18、24，36期
逾期滞纳金 本金金额*0.18%逾期天数 月综合费率 贷款金额*（1.68%-1.98%）
扣款失败手续费 每期第一次扣款失败收取20元 违约金 已还款期限≤贷款期限的一半时，费率=贷款本金*4%；已还款期限>贷款期限的一半时，费率=贷款本金*3%
违约金 已还款期限≤贷款期限的一半时，费率=贷款本金*4%；已还款期限>贷款期限的一半时，费率=贷款本金*3%
基本资料（详细要求见资料要求表）
1、借款申请表 2、征信授权书
3、身份证明 4、银行流水
5、住址证明 6、房产证明（非必要项）
基本条件
年龄 【24，58】周岁 国籍 中国大陆合法公民
收入形式 代发工资 收入要求 A类 ≥2000元
B类、C类 ≥3000元
深圳地区所有客户 ≥4000元
禁贷行业/职业 详见《禁贷行业及职业》 受理区域 详见《营业部受理范围划分》
社保/公积金 必须要有其中一项
必要条件 1、征信查询次数：近一个月查询次数>7次，近3个月查询次数>15次，近6个月查询次数≥20次，满足其中2项拒绝申请
2、执行中案件(包括开庭公告)和已结案拒绝申请；执行完毕案件，且时间<3年拒绝受理。
3、A类客户代发在【1500，1999】之间的客户要求社保或公积金缴纳基数达到2000元，且社保或公积金缴纳记录≥1年
审批额度
增额条件 借款人名下有商品房(商住两用)上浮30%；社保或公积金累计缴纳【13，36】个月或【37，∞】个月，根据审批权限上浮，增额条件可叠加
降额条件 信用卡近6个月透支率≥80%，审批上限10万，且审批权限内下浮。同业贷款>5笔，审批上限为6万，且审批权限内下浮。
计算方法 A类授信额度=代发工作*13倍；B类授信额度=代发工作*10倍；C类授信额度=代发工作*7倍
注：终审额度在上述计算方法上±区间浮动【0%，30%】；A/B类单位员工从事简单体力劳动的按C类客户受理。
