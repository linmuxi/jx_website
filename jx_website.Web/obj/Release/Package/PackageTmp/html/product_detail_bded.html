保单e代 产品说明书
基本特征
客户群体 购买了人寿保险且符合本司相关规定的受薪者或个体工商户 贷款额度 3万-15万
贷款期限 12、15、18、24期 月综合费率 贷款金额*（1.8%-2.3%）
放款手续费 贷款金额*3% 扣款失败手续费 每期第一次扣款失败收取20元
逾期滞纳金 本金金额*0.18%*逾期天数 违约金 已还款期限≤贷款期限的一半时，费率=贷款本金*4%；已还款期限>贷款期限的一半时，费率=贷款本金*3%
基本资料（详细要求见资料要求表）
1、申请表及征信授权书(已婚提供配偶身份证) 2、身份证明
3、保单证明 4、住址证明
5、房产证明、车辆证明、其他证明（非必须项）
基本条件
国籍 中国大陆合法公民 年龄 24-58周岁
现单位工作时长 受薪类客户≥6个月 现单位工作时长 生意类客户≥6个月
企业注册时长 生意类客户≥6个月 股份占比 生意类客户<20%
禁贷行业 详见《禁贷行业及职业》 受理区域 详见《营业部受理范围划分》
保单要求 1、适用于所有保险公司（以审批部核查为准）；
2、购买的必须是商业性质的寿险（意外险、健康险不受理，寿险万能型不受理），如是组合险，则必须含有分红型，且投保人为申请人本人；
3、目前缴费状态需为缴费有效且在缴费期间（逾期、无效或趸交状态不认可）；
4、产品缴费期限大于等于3年，保单已缴费大于等于6个月（从保险生效日期开始计算）；
5、保单月缴保费大于或等于200元（最多3份保单累计大于或等于201元即可）。
6、保单非本省内缴纳的不在受内范围之内
必要条件 1、征信查询次数：近一个月查询次数>7次，近3个月查询次数>15次，近6个月查询次数≥20次，满足其中2项拒绝申请
2、执行中案件(包括开庭公告)和已结案拒绝申请；执行完毕案件，且时间<3年拒绝受理。
3、同行业或保险行业从业人员，暂不受理进行申请。
审批额度
增额条件 借款人名下有商品房(商住两用)上浮30%；社保或公积金累计缴纳【13，36】个月或【37，∞】个月，根据审批权限上浮，增额条件可叠加
降额条件 信用卡近6个月透支率≥80%，审批上限10万，且审批权限内下浮。同业贷款&gt;5笔，审批上限为6万，且审批权限内下浮。
